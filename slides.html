So What's the Problem with DNS?


  • repressive governments routinely attack DNS

  • bad guys easily exploit gaping holes in DNS

  • the weakest link in Internet infrastructure

ICANN Awareness


The hacker inside me likes decentralization architecture. It could be argue that much of the "political problems" we have today derives from the centralized nature of the DNS with the root. So technology like namecoins or other decentralized identifier system intricates me. - James Seng

      from March 2014 draft from ICANN's Technology Innovation Panel

https://www.icann.org/en/about/planning/strategic-engagement/identifier-technology/report-21feb14-en.pdf

Centralization of DNS Creates Vulnerabilities


  • central attack vectors, e.g. rootservers

  • one private key to sign root certs, heavily guarded

  • registry operators become targets for attack

Decentralized DNS Provides Political Benefits


  • free from political corruption

  • censorship resistant

  • domains cannot be seized

  • more resilient when under attack

A Few Notes About Bitcoin


  • a currency, a payment system, a protocol

  • central theme: remove need for trusted 3rd parties

  • working example of blockchain technology

Blockchain Basics


  • distributed consensus - every node on the network agrees

  • immutable transaction records

  • arbitrary data can be inserted

Namecoin Basics


  • the original clone of bitcoin

  • not a financial system per se

  • instructions added to reserve and update names
